= ONTAP FLI 마이그레이션에 대해 알아보십시오
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이 섹션에서는 Foreign LUN Import(FLI) 가져오기 워크플로에 대해 설명하며, 외부 LUN 검색부터 가져오기 실행, 모니터링 및 최종 정리까지 전체 수명 주기를 다룹니다. ONTAP가 기존 백엔드 연결(FC 또는 iSCSI)을 사용하여 호스트에서 프로비저닝 해제되고 ONTAP 이니시에이터에만 매핑된 외부 LUN을 안전하게 검색하는 방법을 설명하여 이중 프레젠테이션 위험이 없도록 합니다. 그런 다음 워크플로는 외부 디스크가 ONTAP 내에서 식별되고 표시되는 방법, 대상 LUN이 소스 지오메트리와 일치하도록 프로비저닝되는 방법, 소스 및 대상 장치를 연결하는 영구 가져오기 관계가 생성되는 방법을 자세히 설명합니다.

이 섹션에서는 LUN 상태 및 타이밍 관점에서 오프라인 및 온라인 FLI 간의 운영상 차이점을 명확히 설명하고, 기본 ONTAP 가져오기 명령은 두 모드 모두에서 동일하다는 점을 강조합니다. 가져오기 시작, 모니터링, 속도 조절(일시 중지/재개) 및 완료 방법과 필요한 경우 데이터 무결성을 검증하기 위한 선택적 검증 방법을 설명합니다. 마지막으로 데이터 복사가 완료되면 가져오기 관계를 안전하게 제거하고 대상 LUN을 완전한 운영 환경으로 전환하는 단계를 설명하며 FLI 워크플로를 마무리합니다.

== 마이그레이션 후 - 온라인 FLI

LUN 가져오기를 삭제한 후 LUN에 `-space-reserve disabled` 및 `-space-allocation enabled`가 설정되어 있는지 확인하십시오. 설정되어 있지 않으면 적절하게 수정하십시오.

SCSI UNMAP 기능이 올바르게 작동하려면 지원되는 소프트웨어 버전을 확인하기 위해 공급업체 설명서를 참조하십시오. 이 기능을 사용하려면 Veritas VxVM + VxFS 조합이 필요합니다.

[cols="4"]
|===
| 파일 시스템 | 블록 디바이스 | LVM(FS) | Sanboot 


| 1단계: 매핑된 모든 HP-UX 호스트에서 스토리지를 다시 검색합니다 | 1단계: 매핑된 모든 HP-UX 호스트에서 스토리지를 다시 스캔합니다. | 1단계: 매핑된 모든 HP-UX 호스트에서 스토리지를 다시 검색합니다 | 1단계: 매핑된 모든 HP-UX 호스트에서 스토리지를 다시 검색합니다 
|===
.단계
. 스토리지 디바이스를 식별합니다.
+
[source, cli]
----
ioscan -fnC disk
insf -e
sanlun lun show all
sanlun lun show all -pv
sanlun lun show -wwpn
----
. VG를 가져오고 VG를 온라인으로 전환합니다.
+
[source, cli]
----
vgimport /dev/vg_name /dev/disk/disk_name
vgchange -a y /dev/vg_name
----
. 볼륨 그룹과 파일 시스템을 식별합니다.
+
[source, cli]
----
vgdisplay -v /dev/vg_name
vgdisplay /dev/vg_name
lvdisplay /dev/vg_name/lv_name
----
. 파일 시스템을 마운트합니다.
+
[source, cli]
----
bdf
mount /dev/vg_name/vol_name /mount_point
----

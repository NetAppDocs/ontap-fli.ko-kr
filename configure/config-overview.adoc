= ONTAP FLI 구성에 대해 알아보십시오
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이 섹션에서는 파이버 채널(FC) 및 iSCSI 백엔드 연결을 모두 사용하는 외부 LUN 가져오기(FLI)의 구성 및 유효성 검사 프레임워크를 설명합니다. LUN 검색 및 가져오기 과정에서 ONTAP에서 외부 스토리지 어레이로의 안정적이고 이중화된 액세스를 보장하는 데 중점을 두며, 호스트 액세스 경로와의 명확한 분리를 유지합니다. 이 섹션에서는 권장 SAN 설계 원칙(듀얼 패브릭 FC, 이니시에이터/타겟 포트 사용, 영역 설정 모범 사례), ONTAP의 어댑터 구성 요구 사항, 그리고 가져오기 시작 전에 외부 LUN이 정상적으로 표시되고 작동하는지 확인하는 운영 점검 사항을 단계별로 안내합니다.

또한, 이 문서는 ONTAP 소프트웨어 iSCSI 이니시에이터를 클러스터 간 LIF(IC LIF)에 바인딩하여 명시적 세션 생성(검색 세션 없음), 복원력을 위한 풀 메시 연결, 네트워크 연결 가능성 및 세션 상태에 대한 엄격한 검증을 강조하는 iSCSI 백엔드 FLI 모델을 자세히 설명합니다. FC 및 iSCSI 시나리오 모두에서 목표는 FLI 필수 요건을 충족하고, 모든 경로에서 일관된 외부 LUN 검색을 가능하게 하며, 검색 및 가져오기 실행 중 발생하는 문제를 해결하는 데 필요한 계측(통계 및 가시성)을 제공하는 결정적이고 지원 가능한 백엔드 전송을 구축하는 것입니다.

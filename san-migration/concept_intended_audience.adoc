= 대상
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이 콘텐츠를 사용하면 외부 어레이에서 ONTAP로 데이터를 마이그레이션하거나 NetApp 7-Mode 어레이에서 ONTAP로 LUN을 전환하는 데 도움이 됩니다.

조닝, LUN 마스킹, LUN을 마이그레이션해야 하는 호스트 운영 체제, ONTAP 및 타사 소스 스토리지를 비롯한 일반적인 SAN 개념 및 프로세스를 숙지해야 합니다.
